HTML Quiz

1. What does HTML stand for?

Correct Answer: A) Hyper Text Markup Language

2. What is the correct HTML element for inserting a line break?

Correct Answer: A) <br>

3. Which HTML tag is used to define an unordered list?

Correct Answer: A) <ul>

4. Which HTML tag is used to define a hyperlink?

Correct Answer: A) <a>

5. Which HTML attribute is used to define inline styles?

Correct Answer: A) style

6. Which HTML tag is used to define a table row?

Correct Answer: A) <tr>

7. Which HTML tag is used to define a header cell in a table?

Correct Answer: A) <th>

8. Which HTML tag is used to define a paragraph?

Correct Answer: A) <p>

9. Which HTML tag is used to define an image?

Correct Answer: A) <img>

10. Which HTML tag is used to define the document's body?

Correct Answer: A) <body>
