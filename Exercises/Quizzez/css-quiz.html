CSS Quiz

1. What does CSS stand for?

Correct Answer: A) Cascading Style Sheets

2. Which property is used to change the text color of an element?

Correct Answer: C) color

3. Which CSS property is used to control the spacing between lines of text?

Correct Answer: A) line-height

4. Which CSS property is used to make text bold?

Correct Answer: A) font-weight

5. Which CSS property is used to add shadows to elements?

Correct Answer: B) box-shadow

6. Which CSS property is used to control the size of an element's font?

Correct Answer: A) font-size

7. Which CSS property is used to align text horizontally?

Correct Answer: A) text-align

8. Which CSS property is used to set the background color of an element?

Correct Answer: A) background-color

9. Which CSS property is used to make an element's text italic?

Correct Answer: A) font-style

10. Which CSS property is used to control the space between the border and content of an element?

Correct Answer: B) padding
